Launch Highlights

The headline changes people will actually notice

These notes condense the larger private changelog into a shorter public-facing update that makes sense to future users.

Modernized app experience

Dreamspell has been rebuilt around a cleaner, more modern interface that makes daily kin lookup, calendar study, and reflection easier to move through.

Offline-first reference flow

The core app experience is built around bundled data and local calculation, so the main study surfaces do not depend on a live connection.

Private on-device journal

Journal entries stay local to the device in the current version, keeping reflective notes inside the app without requiring an account.

Broader launch support

The relaunch scope includes iPhone and iPad layouts, English and Spanish support, and refreshed learning surfaces across the app.

Current Notes

What’s new in the current release track

These notes reflect the current state of the app and stay aligned with the in-repo metadata package and launch materials.

April 11, 2026 launch-readiness notes

  • Dreamspell now centers the experience on Today, Calendar, Kin Explorer, Journal, and Resources.
  • The app includes offline Dreamspell reference content for tones, seals, moon definitions, and related study material.
  • The Journal feature supports searchable reflections tied to dates and kin while keeping entries on device.
  • The app supports both iPhone and iPad and includes English and Spanish localization for the current release path.
  • The current launch scope keeps contacts import inactive and does not require an account.
  • Submission assets now include updated screenshots, public support/privacy pages, and App Store-ready metadata guidance.

Short App Store version

Dreamspell has been fully rebuilt with a modern interface, updated date tools, offline reference content, and a private on-device journal.

What is part of the current promise

  • daily kin lookup and calendar study
  • Kin Explorer for tones, seals, and combinations
  • local journaling tied to dates and kin
  • bundled reference content and learning material
  • iPhone and iPad support
  • English and Spanish localization

What is not being promised at launch

  • no live App Store product URL yet
  • no active contacts import in the current version
  • no account system for the current feature set
  • no server-backed journal sync in the current version
