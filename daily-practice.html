How To Use It

Four practical steps

This is designed for real life, not an idealized study marathon.

Step 1

Check Today

Open the Dashboard and get today’s kin, date, phase, and reflection context quickly.

Step 2

Explore The Cycle

Use Calendar or Kin Explorer when you want to see how today fits inside the larger rhythm.

Step 3

Read Context

Use the in-app course, tones, seals, or other reference screens when you want the symbols explained better.

Step 4

Keep A Note

Write a local journal note if the day, pattern, or reflection feels worth returning to later.
