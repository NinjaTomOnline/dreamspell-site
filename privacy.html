Dreamspell Privacy Policy

Privacy Policy

Last updated: April 11, 2026. This policy describes the current Dreamspell app version for iPhone and iPad and will be revised if the app’s data-handling practices change in a future release.

What information the current app uses

  • Dreamspell reference data is bundled inside the app and used locally on device.
  • Journal entries are stored locally on the device using Apple system persistence.
  • The app does not require an account to use the current feature set.
  • The current version is described as not using third-party analytics, third-party advertising, or server-backed journal storage.

What the current app does not do

  • It does not require account creation to use the shipped feature set.
  • It does not include active contacts import in the current version.
  • It does not describe journal upload to the developer in the current version.
  • It does not describe selling user data or using app data for cross-app tracking.

Tracking, Sharing, And Retention

How the current version handles user data

This policy stays aligned with the documented behavior of the current app version and avoids promising features that are not live.

Tracking and advertising

Dreamspell is described as not using app data for tracking across apps or websites, not including third-party advertising, and not selling user data in the current version.

Data sharing

The current version is described as not transmitting journal content or Dreamspell calculation data to the developer, and not sharing journal entries or Dreamspell readings with third parties.

Data retention

Journal entries remain on the device until the user edits or deletes them, or removes the app and its local data. Bundled reference data remains part of the installed app package.

Permissions And Future Changes

What to expect from the current release

If a future version changes permissions or data flow, this policy should be updated before that version is released.

Contacts access

The app includes usage-description copy for a possible future opt-in contacts feature, but that feature is not active in the current version and is not part of the current public promise.

Policy updates

If Dreamspell’s privacy practices change in a future release, this page and the related App Store privacy disclosures should be updated before that version is distributed.

Children

Dreamspell is not marketed as a Made for Kids app in the current release materials.
