At Launch

What Dreamspell includes today

This summary stays limited to features already reflected in the current app materials and testable product scope.

Today

See the current Gregorian date, daily kin, lunar phase, and affirmation in one place.

Calendar

Browse the 13 Moon cycle and inspect the Tzolkin as part of daily study.

Kin Explorer

Combine galactic tones and solar seals, inspect kin more closely, and explore group-kin reflection tools.

Journal And Resources

Keep local reflections tied to dates and kin, and browse bundled reference material inside the app.

What Dreamspell does not require

  • no account creation for the current feature set
  • no always-on internet connection for core use
  • no active contacts import in the current version
  • no server-backed journal storage in the current version
