Common support topics

  • Does Dreamspell need internet access? No. The current version is designed around bundled reference data and local computation.
  • Where are journal entries stored? On the device in the current version.
  • Does the app require an account? No account is required for the current feature set.
  • Does the app import contacts? Not in the current version.
  • Is Dreamspell available on iPad? Yes. The app supports both iPhone and iPad.

Good details to include in a support request

  • your device model and iOS version
  • whether the issue happened on iPhone or iPad
  • the app language you were using
  • which area was affected: Dashboard, Calendar, Kin Explorer, Journal, or Resources
  • the date or kin you were trying to inspect, if relevant
  • a screenshot or short description of what happened instead of what you expected

Report a problem from inside the app

If a support build shows the Diagnostics card, open Resources, expand Settings, tap Copy Diagnostics, and paste that copied text into your email. The diagnostics payload helps identify the build, device, language, startup phase, current tab, and recent launch events. It does not include journal body text.

In the normal App Store build, Diagnostics may be hidden. If you do not see it, just include your device model, iOS version, affected screen, and the shortest steps that reproduce the issue.

Troubleshooting

Practical checks by app area

If your question is about how a screen behaves, these quick checks usually save time before you send a report.

Dashboard

If today’s signature looks unexpected, note the Gregorian date shown on screen and include it in your message so the date conversion path can be checked.

Calendar

If a day or kin selection feels wrong, mention whether you were in the 13 Moon view or the Tzolkin view and which date or kin you tapped.

Kin Explorer

If a combination or blend result is confusing, include the tone and seal you selected and whether you were reviewing favorites, recent kin, or a group kin blend.

Journal

If a note is missing, first confirm you are on the same device and app installation where you created it. Journal entries stay local in the current version.

Resources

If a glossary or learning view does not open as expected, mention the exact card or lesson you tapped so the navigation flow can be reproduced.

Localization

If you spot a translation issue, tell us whether you were using English or Spanish and include the screen and wording you saw.
